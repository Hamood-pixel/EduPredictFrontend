<h3>🚀 Project Overview</h3>
<p>
  EduPredict is a high-performance web interface designed to bridge the gap between complex <b>Machine Learning</b> models and educational administrators. It utilizes a React-based frontend to collect student data points and provides instant predictions regarding their academic trajectory.
</p>

<h3>✨ Key Features</h3>
<ul>
  <li><b>Dropout Prediction:</b> Leverages the UCI "Student Dropout" dataset to predict outcomes with high accuracy.</li>
  <li><b>Vite-Powered Speed:</b> Optimized build times and Hot Module Replacement (HMR).</li>
  <li><b>Containerized:</b> Includes a <code>Dockerfile</code> for easy deployment and environment parity.</li>
  <li><b>Modern UI:</b> Clean interface for entering demographic and socioeconomic parameters.</li>
</ul>

<h3>⚙️ Setup & Installation</h3>

<p><b>1. Clone the repository:</b></p>
<pre><code>git clone https://github.com/YourUsername/edupredict-frontend.git</code></pre>

<p><b>2. Install and Run:</b></p>
<pre><code>npm install
npm run dev</code></pre>

<p><b>3. Run with Docker:</b></p>
<pre><code>docker build -t edupredict-app .
docker run -p 5173:5173 edupredict-app</code></pre>

<p>
  This project utilizes the <i>"Predict students' dropout and academic success"</i> dataset from the <b>UCI Machine Learning Repository</b>. 
  The data includes academic records, demographics, and social-economic factors at the time of enrollment.
